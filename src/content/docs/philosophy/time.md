In ParaDigi Universe, time is not treated as a mystical flow. It is an ordering tool used by observers to organize change.

### Matter, Space, and Change
The universe consists of **matter** and **space**. Change is a property of matter in space.

### From Change to Event
1. **Change:** The objective movement of matter.
2. **Event:** A cognitive "package" created by an observer to simplify complex changes.
3. **Time:** The measurement of the partial order of these events.

### Why this matters for the protocol
Because a decentralized network has no universal clock, the protocol defines identity through a **total order** of events.

- if a publisher creates conflicting next events, the identity loses coherence
- in peer-to-peer space, "now" is only the edge of the information currently available to an observer
