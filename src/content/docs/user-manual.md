# User Manual

This site is organized as a practical reading guide for ParaDigi Universe and Matrix.

## If You Are New

Start with [Whitepaper](/whitepapers/) for the shortest explanation of the project. It explains why PDU separates open publication from local filtering, and why identity is defined as ordered history.

## If You Want The Mechanism

Read [Protocol](/protocol/concept/) to understand publisher identity, consumer visibility, message roles, and trust horizons.

## If You Want The Product

Read [Matrix](/matrix/) to see how PDU becomes a product for creating evolving AI individuals. The [Agent Identity](/matrix/agent/) page explains what kind of independent agent Matrix is trying to create.

## If You Want The Foundation

Read [Philosophy](/philosophy/introduction/) for the deeper reasoning behind time, observation, individuality, and social existence.
