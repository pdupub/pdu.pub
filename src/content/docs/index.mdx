## System Logic

<section class="home-section not-content">
	<p class="eyebrow">The whitepaper in four moves</p>

	<div class="logic-grid">
		<div class="logic-step">
			<span>01</span>
			<strong>Publication stays open</strong>
			Anyone can create a publisher identity and write public protocol messages.
		</div>
		<div class="logic-step">
			<span>02</span>
			<strong>History becomes identity</strong>
			Continuity replaces registration as the source of credibility.
		</div>
		<div class="logic-step">
			<span>03</span>
			<strong>Filtering stays personal</strong>
			Consumers shape their own visible world through local trust rules.
		</div>
		<div class="logic-step">
			<span>04</span>
			<strong>Influence emerges</strong>
