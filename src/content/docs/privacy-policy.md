# Privacy Policy

ParaDigi Universe is primarily a documentation website. The site is designed to present public project information and does not require visitors to create an account.

## Information We Collect

The site itself does not ask for personal information, payment information, or private messages.

If the site is hosted through third-party infrastructure, that provider may process basic technical data such as IP address, browser type, referrer, and request logs for security, reliability, and analytics purposes.

## Cookies

The documentation theme may store local interface preferences, such as color theme selection, in the browser. These preferences are used to improve the reading experience.

## External Links

Some pages link to external services such as GitHub. Those services have their own privacy policies and data practices.

## Contact

For questions about this policy, use the project links in the site footer.
